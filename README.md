# News Search App

A simple news search application with three independent search columns.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Open your browser and go to: `http://localhost:3000`

## Features

- Three independent search columns
- Search for news articles by keyword
- Displays article title, image, description, and link
- Responsive design for different screen sizes

## Usage

- Type a search term in any column's input field
- Click the Search button or press Enter
- View the search results displayed below
- Click "Read more" to open the original article
